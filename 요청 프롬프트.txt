server.js에 텔레그램 연동을 추가해줘.
토큰과 chat_id는 .env에서 읽어와.

## 로컬 → 텔레그램 알림 (단방향)
브라우저에서 아래 동작 시 텔레그램으로 알림 전송:
- 추가 버튼 클릭 → "✅ [항목명] 추가됐어요"
- 완료 버튼 클릭 → "☑️ [항목명] 완료됐어요"
- 삭제 버튼 클릭 → 알림 없음

## 텔레그램 → 로컬 처리 (양방향)
텔레그램에서 아래 명령어 입력 시 서버에서 처리:
- "추가 장보기" → 장보기 항목 추가
- "변경 1 마트가기" → 1번 항목 내용을 마트가기로 변경
- "완료 1" → 1번 항목 완료 처리
- "삭제 1" → 1번 항목 삭제
- "목록" → 현재 Todo 목록 전송

## 조건
- 텔레그램 수신은 polling 방식으로 구현
- 존재하지 않는 번호 입력 시 "해당 항목을 찾을 수 없어요" 응답
- 명령어 형식이 잘못됐을 때 "올바른 명령어를 입력해주세요" 응답
